Rota Prototype

About This Prototype

ChalkboardRota helps you build a weekly intervention timetable quickly, accurately, and without sharing pupil data.

Please only upload sample or dummy data.

In the full version, you’ll upload your class list and timetable, tag when support is available, and the tool will create an anonymised plan. In this prototype, there’s a built-in sample timetable so you can see the workflow in action. The AI only sees pseudonyms; real names stay on your device and are added back in at the end.

This version is for demonstration only.

It is an early prototype — expect rough edges — but it shows the full workflow. Try it out and see how the rota adjusts to your class, timetable, and staffing.

A full, secure, school-ready version is in development.


How to test this prototype

  1. Load the sample class list.
  2. Click Check for names → then Anonymise.
  3. Load the sample timetable.
  4. Click a session to tag how many adults are available.
  5. Press Generate intervention plan.

Use Show real names to toggle between pseudonyms and real names.

How to get your timetable into this tool

1. Open your timetable in Excel, Google Sheets or similar.

2. Make sure it has one row per lesson, with columns called Day, Start, End and Label.

3. If your timetable is currently in a Word document, copy the table into a blank spreadsheet first, tidy up the headings, then save it as a simple text-table file.

4. Upload that saved file here. Each row should look something like: Mon | 09:00 | 09:20 | Assembly.

You can always fall back to the built-in sample timetable while you’re testing.
